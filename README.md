Ecommerce Frontend

Frontend for a modern Ecommerce platform built using React, Vite & Tailwind CSS.

Live Demo
View live at: https://ecommerce-frontend-theta-three.vercel.app

Features

- Responsive product listing and product detail pages
- Add to cart and remove from cart
- Cart persistence using local storage or context
- Product sorting
- Checkout UI (UI only, backend integration optional)

Tech Stack

-  React
-  Vite
-  Tailwind CSS
-  Axios (for API calls)
-  React Router v6
